# `sun50i-a64-gapdc-iio`

[Vasily Khoruzhick][anarsoul]'s Allwinner thermal sensor driver tweaked into
an out-of-tree kernel module that exposes `sun50i-a64-ths` compatibility only.

The driver [has been mainlined in Linux 5.6][torvalds/linux@59f5e9b].
Subsequently, this project is no longer maintained.
