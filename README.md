# `sun50i-a64-gapdc-iio`

[Vasily Khoruzhick][anarsoul]'s Allwinner A64 thermal sensor driver as
an out-of-tree Linux kernel module.

The driver [has since been mainlined in Linux 5.6][torvalds/linux@59f5e9b].
